# ReClone
mak clone beryy good.

A 2d game about making clones to solve puzzles. thats about it..

## Controls
Use A and D to walk and Space to jump. Use R to reset, C to make a clone and ESC to quit to menu.
Also you can left click and drag to move your camera around.

## Levels

14 Levels in the game. They are not based on hardness but mechanics to first few are easier then its kinda random.
